# JPackage Maven Plugin

Maven plugin for [jpackage](https://docs.oracle.com/en/java/javase/25/docs/specs/man/jpackage.html).

```xml
<plugin>
    <groupId>org.panteleyev</groupId>
    <artifactId>jpackage-maven-plugin</artifactId>
    <version>1.7.2</version>
</plugin>
```

[Plugin Documentation](https://www.panteleyev.org/javadoc/jpackage-maven-plugin/)

[Packaging Tool User's Guide](https://docs.oracle.com/en/java/javase/25/jpackage/packaging-tool-user-guide.pdf)
