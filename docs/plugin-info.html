Plugin Documentation

This report describes goals, parameters details, requirements and sample usage of this plugin.

Goals

Goals available for this plugin:

Goal Description
jpackage:help Display help information on jpackage-maven-plugin.
Call mvn jpackage:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
jpackage:jpackage

Generates application package.

Each plugin parameter defines jpackage option. For detailed information about these options please refer to Packaging Tool User's Guide

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.9.7
JDK 1.8

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>org.panteleyev</groupId>
          <artifactId>jpackage-maven-plugin</artifactId>
          <version>1.6.6</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>org.panteleyev</groupId>
        <artifactId>jpackage-maven-plugin</artifactId>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
