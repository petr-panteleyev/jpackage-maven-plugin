JPackage Maven Plugin

Maven plugin for jpackage.

Goals Overview

  • jpackage:jpackage generates application package.

Usage

General instructions on how to use the JPackage Maven Plugin can be found on the usage page. Some more specific use cases are described in the examples given below.

If you feel like the plugin is missing a feature or has a defect, you can fill a feature request or bug report in our issue management system. When creating a new issue, please provide a comprehensive description of your concern. Especially for fixing bugs it is crucial that the developers can reproduce your problem. For this reason, entire debug logs, POMs or most preferably little demo projects attached to the issue are very much appreciated.
